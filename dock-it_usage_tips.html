Getting Started

📅 Daily Tasks

  1. Type in the Add new task... field at the bottom of the daily list and tap Return or the + button
  2. Swipe left or right to move between days, or tap the calendar icon in the header to jump to any date
  3. Tap the circle to the left of any task to check it off
  4. Tap the task name to edit it inline

📋 Static Lists

  1. Tap + in the bottom banner to create a new static list
  2. Swipe left or right in the lower panel to move between lists
  3. Tap the list title to rename it inline
  4. Tap the ··· menu in the list header for Clean, Reset, Export, and Delete options

🔲 Bottom Banner

The bottom banner gives you quick access to six controls:

  • ⚙️ Settings — app preferences
  • 👁 Hide/Show — toggle completed task visibility
  • ↑↓ Sort — cycle sort: none → top → bottom
  • ⊡⊡ Reorder — drag static lists to reorder
  • + New List — create a new static list
  • 🔍 Search — find any task

Features In Depth

Tasks & Organization

☑️ Subtasks

  1. Tap ··· on any task and choose Add Subtask
  2. Type the subtask name and tap Return to add more
  3. Subtasks appear indented below their parent
  4. Completing all subtasks automatically checks the parent
  5. Unchecking any subtask automatically unchecks the parent

⭐ High Priority List

  1. Tap ··· on any task → Mark High Priority
  2. A live copy appears in your HP list with an origin indicator
  3. Check it off from either location — both sync instantly
  4. Tap the origin indicator on the HP copy to jump back to the source list or date
  5. To remove: tap ··· → Remove High Priority

↕️ Manual Reordering

  1. Long-press the ≡ handle on the right of any task
  2. Drag the task to its new position
  3. Release to drop — sort order is saved automatically
  4. If completion sort is active, dragging cancels it and restores manual order

👁 Hide & Sort Completed

  • Tap the eye icon in the bottom banner to hide completed tasks from view
  • Tap the ↑↓ arrows to sort completed tasks to the top or bottom of the list — tap again to cycle through states
  • Both settings apply globally across all daily and static lists
  • Settings are preserved when you swipe between days or lists

Moving & Copying Tasks

✨ Move To / Copy To

  1. Tap ··· on any task → Move To... or Copy To...
  2. Choose a date option — options adapt based on your currently viewed date
  3. Or choose Static List... to move or copy to any static list
  4. Moving removes the original; copying leaves it in place
  5. Subtasks are carried along automatically

📋 Moving to a Static List

  1. Tap ··· → Move To... or Copy To...
  2. Tap Static List...
  3. Choose any list from the picker — your current list is excluded automatically
  4. Or tap Add New List... to create a new list and place the task in it immediately

Repeating Tasks

🔄 Setting Up a Repeat

  1. Tap ··· on any daily task → Add Repeat
  2. Choose a frequency — Daily, Weekdays, Weekly, Monthly, or Yearly (defaults to Weekly)
  3. Set start and end dates
  4. All instances are generated immediately — scroll ahead to see them

🔄 Managing Repeats

  • Edit Repeat — tap ··· on any instance to adjust the end date
  • Delete This Instance — removes just that one occurrence
  • Delete (on the original) — removes the full sequence
  • Daily and weekday repeats with long durations may affect app performance — consider limiting the end date or using a static list instead

Widget & Search

📱 Widget Setup

  1. Long-press your iPhone home screen → tap +
  2. Search for DOck-IT and choose a size: small, medium, or large
  3. In the app, go to Settings → Configure Widget
  4. Choose which list to display and whether to show completed tasks
  5. Tap the widget to jump directly to that list in the app

🔍 Search

  1. Tap the magnifying glass in the bottom banner
  2. Type any keyword — matches appear highlighted across all lists and dates
  3. Use the arrows to jump between results
  4. Search covers your full task history, not just the current date
  5. Tap the × to close search and return to the current view

📤 Export a Static List

  1. Navigate to the static list you want to share
  2. Tap the ··· menu in the list header
  3. Choose Export List
  4. Select any app to share to — Messages, Notes, Mail, and more

📆 Add Task to Calendar

  1. Tap ··· on any daily task
  2. Choose Add to Calendar
  3. Review and edit the event details in the calendar editor
  4. Tap Add to save to your iOS Calendar

Static List Management

📋 List Controls

  • Rename — tap the list title to edit it inline
  • Clean List — removes all completed tasks (tap ··· in the list header)
  • Reset List — unchecks all tasks, returning the list to a fresh state
  • Reorder Lists — tap the reorder icon in the bottom banner, then drag lists into order
  • Delete List — removes the list and all its tasks permanently

📊 App Statistics

Open Settings (gear icon, bottom left) and scroll to the App Data section to see:

  • Total daily task count
  • Total static list task count
  • Data store file size on device

⚠️ Note for users updating from v1

DOck-IT v2 uses an improved repeat task system. When you first open the app after updating, your existing repeat sequences will be automatically converted — previously generated instances are preserved as regular tasks, and you'll be prompted to recreate any repeat sequences you'd like to continue. This is a one-time process.
