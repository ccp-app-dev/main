Key Features

📅

Daily Task Management

Swipe through dates effortlessly. Navigate with arrows or jump to any date with the calendar picker. The year is shown automatically when viewing past or future years.

📋

Static Lists

Create unlimited persistent lists for Work, Travel, Shopping, or anything else. Reorder, rename, clean, reset, and export lists to share anywhere.

🔄

Repeating Tasks Improved

Set tasks to repeat daily, weekdays only, weekly, monthly, or yearly. All instances are generated upfront so your full sequence is always visible, even far into the future.

☑️

Subtasks New

Break any task into subtasks. A parent task is automatically marked complete when all its subtasks are done, and unchecked when any subtask is unchecked.

⭐

High Priority List New

Mark any task as High Priority to pin a live copy to your dedicated HP list. Check it off from either location — both stay in sync automatically, including subtasks.

📱

Home Screen Widget New

Add a DOck-IT widget to your home screen in small, medium, or large size. Shows today's tasks, your HP list, or any static list. Tap the widget to jump straight to that list in the app.

📤

Export Static Lists New

Share any static list as formatted text via Messages, Notes, Mail, or any app on your device. Perfect for sharing checklists or packing lists with others.

📆

Add to Calendar New

Send any daily task directly to your iOS Calendar. Review and edit event details before saving — full EventKit integration with your existing calendars.

🔍

Powerful Search

Find any task instantly with full-text search and highlighting across all your lists and dates — including tasks far outside your current date view.

✨

Move & Copy Tasks

Move tasks between dates, copy to multiple days, or transfer between lists. Static list tasks can also be copied directly to a specific daily date.

📊

App Statistics New

View your total daily task count, static list task count, and data store size from the Settings menu. Useful for understanding how much data your app is managing.

🎨

Dark Mode & Accessibility

Beautiful light and dark themes. Adjustable text size from 80% to 150%. Optional strikethrough for completed tasks. Auto-capitalize words as you type.

🔒

Privacy First

Your data stays yours. All information stored locally on your device. No accounts, no servers, no tracking.

How To Use

⭐ High Priority

  1. Tap ··· on any task and choose Mark High Priority
  2. A live copy appears in your HP list
  3. Check it off from either location — both sync instantly
  4. Tap the origin indicator on the HP copy to jump back to the source
  5. To remove: tap ··· → Remove High Priority

☑️ Subtasks

  1. Tap ··· on any task and choose Add Subtask
  2. Type the subtask name and tap return to add more
  3. Subtasks appear indented below their parent
  4. Completing all subtasks automatically checks the parent

🔄 Repeating Tasks

  1. Tap ··· on any daily task → Set Repeat
  2. Choose frequency: Daily, Weekdays, Weekly, Monthly, or Yearly
  3. Set a start and end date
  4. All instances are generated immediately
  5. Delete a single instance or the full sequence from any instance's menu

📱 Widget Setup

  1. Long-press your iPhone home screen → tap +
  2. Search for DOck-IT and choose a size
  3. In the app, go to Settings → Configure Widget
  4. Choose which list to display and whether to show completed tasks or subtasks
  5. Tap the widget to open the app directly to that list

📤 Export a Static List

  1. Navigate to any static list
  2. Tap the ··· menu in the list header
  3. Choose Export List
  4. Select any app to share to — Messages, Notes, Mail, and more

📆 Add Task to Calendar

  1. Tap ··· on any daily task
  2. Choose Add to Calendar
  3. Review and edit the event details
  4. Tap Add to save to your iOS Calendar

📋 Static List Functions

  • Clean List — removes all completed tasks from the current list
  • Reset List — unchecks all tasks, returning the list to a fresh state
  • Reorder Lists — drag lists into your preferred order from the list header menu
  • Rename — tap the list title to edit it inline

📊 App Statistics

Open Settings (person icon, bottom left) and scroll to the App Data section to see your total daily task count, static list task count, and data store file size.

⚠️ Note for users updating from v1

DOck-IT v2 uses an improved repeat task system. When you first open the app after updating, your existing repeat sequences will be automatically converted — previously generated instances are preserved as regular tasks, and you'll be prompted to recreate any repeat sequences you'd like to continue. This is a one-time process.

Privacy Policy

DOck-IT is designed with your privacy as the top priority. We believe your tasks and personal information should remain private.

Data Collection

  • No data collection: DOck-IT does not collect, transmit, or store any of your personal information on external servers.
  • Local storage only: All tasks, lists, and settings are stored locally on your device using SwiftData.
  • Widget data: The home screen widget reads task data from a local shared container on your device. No data leaves your device.
  • No analytics: We do not track your usage, behavior, or any analytics.
  • No third-party services: The app does not integrate with any third-party analytics or advertising services.

Permissions

  • Calendar access: Only requested when you use Add to Calendar. Used solely to create events you explicitly initiate. Never accessed in the background.
  • No other permissions required: DOck-IT does not request access to contacts, photos, location, microphone, camera, or any other device data.

Your Rights

  • Data ownership: You own all your data. It never leaves your control.
  • Export: Use Export List to share your static list data in plain text at any time.
  • Delete: Deleting the app removes all local data from your device.
  • No account required: DOck-IT works without creating an account or providing personal information.

If you have any questions about our privacy practices, please contact us at support@dock-it.net
