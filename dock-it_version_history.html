Version History

Version 2.1 July 2026 Latest

New Features

  • Bottom banner redesigned with six quick-access controls: Settings, Hide/Show Completed, Sort Completed, Reorder Lists, Add New List, and Search — all evenly spaced for one-tap access
  • Hide/Show Completed moved to the bottom banner for instant toggle without opening Settings
  • Completed task sort moved to the bottom banner — tap to cycle between no sort, sort to top, and sort to bottom; active direction shown in blue
  • Reorder Lists moved to the bottom banner with a custom icon
  • Add New List moved to the bottom banner
  • Move To / Copy To now supports static lists as a destination — move or copy any task directly into any static list
  • Move To / Copy To from a static list now shows Today or Displayed Day depending on the date currently in view
  • Move To / Copy To from a daily list adapts to context — shows Tomorrow when viewing today, or Today when viewing another date
  • Create New List option added to the Move To / Copy To static list picker — creates a new list and immediately places the task in it
  • Copying a task with subtasks now correctly copies all subtasks to the destination
  • Repeat frequency defaults to Weekly when creating a new repeat
  • Performance warning shown the first time Daily or Weekdays frequency is selected for a repeat task

Improvements

  • Manual task reordering now cancels completion-based sort automatically — dragging a task resets sort to none and preserves the new manual order
  • Fixed spontaneous task resorting caused by completion sort influencing drag index calculations
  • Right chevron in static list header is now hidden when on the last list, matching the behavior of the left chevron on the first list
Version 2.0 June 2026

New Features

  • Subtasks — add sub-items to any task; parent auto-completes when all subtasks are done
  • High Priority list — mark any task as HP to pin a live synced copy to a dedicated HP list
  • Home screen widget — small, medium, and large sizes; shows daily, HP, or any static list; tapping navigates directly to that list in the app
  • Widget auto-refreshes daily so your tasks stay current even when the app is closed
  • Export static lists as formatted text to Messages, Notes, Mail, or any app
  • Add any daily task to your iOS Calendar with full event editing
  • Weekday repeat frequency — repeat on Monday through Friday only
  • Reorder static lists — drag lists into your preferred display order
  • Clean List — remove all completed tasks from the current static list in one tap
  • Reset List — uncheck all tasks in the current static list to start fresh
  • App Data in Settings — view total task counts and data store size
  • Year shown in date header when viewing a year other than the current one

Improvements

  • Repeat task system rebuilt — all instances generated upfront so future tasks are always visible
  • Search now spans full task history, not just the current date range
  • HP list check state syncs bidirectionally including subtask cascading
  • Tapping the origin indicator on an HP copy navigates back to the source list or date
  • HP copy names update in real time when the original is edited
  • Deleting a repeat sequence now correctly removes any associated HP copies

Upgrade Note

  • Existing repeat sequences from v1 are automatically converted on first launch — previously generated instances are kept as regular tasks; sequences can be recreated with the new repeat interface
Version 1.1.1 March 2026

Bug Fixes

  • Fixed calendar Pick Date feature showing incorrect list content after selecting a new date
  • Fixed search navigation not jumping to the correct date when a result was found
Version 1.1 March 2026

New Features

  • Copy To option added to static list items — copy any static task directly to a specific date
  • Auto Capitalize Words setting — automatically capitalizes the first letter of each word as you type
  • End Repeat — stop a repeating task from generating future occurrences from any instance
  • Resume Repeat — re-activate a previously ended repeat sequence
  • Repeat indicator now shown on all repeat instances, not just the original task

Bug Fixes

  • Repeat option removed from static list task menus where it was not applicable
  • Fixed a bug where deleting a list would incorrectly create a new empty list
  • Fixed a bug where copying a task to a future date would briefly show the wrong list content
  • Improved reliability of task reordering in static lists
  • Fixed Move To and Copy To using Pick Date not correctly placing tasks on the selected date
  • Fixed calendar date picker briefly showing incorrect list content while browsing dates
Version 1.0 February 2026 Initial Release
  • Daily task management with date navigation
  • Static lists for persistent tasks
  • Repeating tasks (daily, weekly, monthly, yearly)
  • Move and Copy tasks between dates
  • Full-text search across all lists and dates
  • Dark mode and adjustable text size
